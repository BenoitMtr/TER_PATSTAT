# TER_PATSTAT
## Installer OpenLayers
`cd TER_PATSTAT/Front`

`npm install ol`
## Ajouter les dépendances de développement nécessaires
`npm install --save-dev parcel-bundler`
## Lancer le projet 
`npm start`
## Ouvrir dans le navigateur
 http://localhost:1234/
